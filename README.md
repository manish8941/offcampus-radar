# OffCampus Radar

Verified Tech Hiring Alerts for Students & Freshers.

This is a React + Vite + Tailwind CSS landing website for the OffCampus Radar hiring alerts community.

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## Deployment

The site is configured for GitHub Pages at:

```text
https://manish8941.github.io/offcampus-radar/
```

## Registration Collection
